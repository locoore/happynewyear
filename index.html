SimplyGreen contains of only 5 divisions - page, header, main, righty, content; The page is the main container of the page which is responsible for the page to be center. Also it holds the attribute for the width of the page. Changing it to 100% will make the complete page fluid. This template is also Search Engine Optimised.

This template is tested in IE 6.x, Opera 9.01, Netscape 8. This template looks same exactly in Internet Explorer, Opera, Netscape Browsers.

You can also view the 改變版面配置[ 內容底正手邊 Content at the right side ]

The Styles and Classes

This Template mainly relies on a set of 4 colors

  • #6F7850
  • #929E6A
  • #7C8269
  • #002200

This design is very simple and has only 2 classes to align the images and the rest are just divisions(DIV's) which handle the layout

  • .img-left
  • .img-right

These two classes as they suggest align the images to left or right.
The image to the right is aligned using the img-right and the image above is aligned to left using the class .img-left.


The Side headings are achived using border-top and border-left in the style for <h2>...</h2> tags. so just use a h2 and the style is achived.
